A nervous system reset is used when activation, tension, or overload needs to be interrupted and softened.

It may include:
- direct response breathing
- awareness exercises
- supportive setup changes
- brief settling tools

The aim is not to force the system into instant calm. The aim is to interrupt escalation and create enough steadiness for the next helpful step.

A reset can be useful:
- before a longer practice
- when discomfort or overwhelm appears during practice
- when the body needs support before it can settle into stillness
