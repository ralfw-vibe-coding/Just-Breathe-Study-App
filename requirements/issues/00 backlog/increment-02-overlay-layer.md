# Inkrement 02: Overlay Layer

## Ziel

Der persoenliche User-Layer wird auf die globale `base` gelegt und im Alltag direkt nutzbar.

## Scope

- Favoriten anzeigen, sobald vorhanden
- Favoriten hinzufuegen und entfernen
- Favoriten umordnen
- Tags pro Karte bearbeiten
- Plaintext-Notiz pro Karte bearbeiten
- Overlay-Aenderungen optimistisch im Frontend verarbeiten
- Overlay-Teilbereiche im Hintergrund speichern
- Retry bei fehlgeschlagenem Hintergrund-Speichern
- `last write wins` fuer parallele Aenderungen

## Nicht Teil Dieses Inkrements

- Lernkartei
- semantische Suche
- Chat

## Ergebnis

Ein Nutzer kann Karten persoenlich ueberlagern und die App dadurch als eigenes Wissens- und Abrufsystem verwenden.
