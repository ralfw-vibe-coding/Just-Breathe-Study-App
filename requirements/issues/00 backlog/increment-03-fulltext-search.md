# Inkrement 03: Volltextsuche

## Ziel

Schneller direkter Zugriff auf Karten ueber eine clientseitige Volltextsuche.

## Scope

- Suchfeld oberhalb der `base`, unter den Favoriten
- clientseitige Volltextsuche auf der geladenen `base`
- Suche ueber `title`, `overview`, `details`
- Suchergebnis als Kartenliste
- Uebergang von Trefferliste in Kartenansicht
- Overlay-Tag-Suche bzw. Tag-Filter clientseitig auf dem geladenen `overlay`

## Nicht Teil Dieses Inkrements

- semantische Suche
- Chat

## Ergebnis

Ein Nutzer kann in Sekunden Karten ueber Begriffe finden und direkt oeffnen.
